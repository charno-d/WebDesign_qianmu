# QianmuLaser 企业官网

## 技术栈

本项目是基于以下技术构建的企业官网：

### 前端技术
- HTML5 + CSS3
- JavaScript (ES6)
- jQuery 1.11.0
- Bootstrap 5 (通过bootstrap.min.css和bootstrap.bundle.min.js)
- Font Awesome (通过vendor.css引入)

### 项目结构
- 组件化设计 (navbar.html, footer.html)
- 响应式布局
- 图片懒加载
- AJAX内容加载 (通过load-navbar.js, load-footer.js)

### 构建工具
- Visual Studio Code 开发环境
- Git 版本控制

## 项目特点
- 多语言支持 (中英文)
- 产品展示系统
- 新闻资讯模块
- 技术文档展示
- 响应式设计适配各种设备

## 部署方式
1. 克隆本项目
2. 直接打开index.html即可运行
3. 或部署到任意静态网站托管服务
